# Support

## Getting Help

If you need help with FriendScope, here are the available channels:

- **GitHub Discussions**: for general questions, feedback, and ideas, visit the [Discussions](https://github.com/ChanMeng666/friendscope/discussions) tab.
- **GitHub Issues**: for bug reports and feature requests, [open an issue](https://github.com/ChanMeng666/friendscope/issues).

## Common Questions

**Local dev server won't start**
- Run `npm install` first, then `npm run dev`. The app runs on Next.js.

**My assessment results disappeared**
- FriendScope is privacy-focused and minimizes data retention; results may be stored locally in your browser. Clearing site data will remove them.

**How do I deploy my own copy?**
- See the Cloudflare deployment steps in [CONTRIBUTING.md](CONTRIBUTING.md).

## Scope of Support

This is an open-source project maintained on a volunteer basis. We do our best to respond, but
cannot guarantee response times. Please be patient and respectful.

## Security Issues

For security-related concerns, **do not open a public issue**. Follow the process in
[SECURITY.md](SECURITY.md).
